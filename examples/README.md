Current contents:

- `primitive-composition.json` for the canonical primitive composition map
- `greenfield-blog/` for a minimal editorial consumer proof
- `greenfield-workspace/` for a minimal workflow consumer proof
- `validation/` for an invalid manifest and captured repair output
- `../apps/demo-blog/site/index.html` for the editorial reference output
- `../apps/demo-saas/site/index.html` for the workflow reference output
