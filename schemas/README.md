# Schemas

This directory contains the checked-in export set generated from the real New Alphabet contract bundle.

## Canonical bundle

- `context-bundle-0.1.0.json`

## Split exports

- `foundations.json`
- `flavors.json`
- `primitives.json`
- `components.json`
- `recipes.json`
- `composition-rules.json`
- `state-contracts.json`
- `anti-patterns.json`
- `validation-rules.json`
- `examples.json`
- `prompt-intents.json`

## JSON schema documents

- `contract-bundle.schema.json`
- `foundations.schema.json`
- `flavors.schema.json`
- `primitives.schema.json`
- `components.schema.json`
- `recipes.schema.json`
- `project.schema.json`

These files are deterministic splits of the same exported context so coding sessions and tooling can load only the sections they need.
